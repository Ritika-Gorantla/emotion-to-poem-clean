# Emotion to Poem 🌸🖋️

A web app that turns your emotional text into beautiful poetry using OpenAI GPT-4.

## 🧠 Tech Stack
- React (Vite) + TailwindCSS
- Node.js + Express
- OpenAI API (GPT-4)

## 🛠️ Setup

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
touch .env    # Add your OpenAI key inside
node index.js
```

## 📦 Folder Structure
- `client/` – React + Tailwind UI
- `server/` – Express + OpenAI API
